Este proyecto tiene como finalidad desarrollar un sistema software capaz de llevar a cabo la venta de boletos de Cine por medio de una reserva hecha por el Cliente donde el sistema será capaz de decirle la disponibilidad de boletos de una película y un horario determinado. Además, el sistema tendrá una gestión de empleados y clientes pertenecientes a la institución, donde los empleados podrán gestionar las películas, salas, promociones y cartelera, además podrán confirmar las reservas hechas por los clientes.
